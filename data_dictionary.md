# Data Dictionary

Definitions for every released data file. Source data: TCGA via cBioPortal PanCancer
Atlas 2018 (CC0) and TCGA primary publications (UCSC Xena, Broad Firehose GDAC).
Patient identifiers are public 12-character TCGA barcodes (e.g. `TCGA-OR-A5J1`).

## `data/labels/<cohort>_labels.parquet`

One row per patient. Columns vary by cohort (each cohort carries its own driver-mutation
panel); the common schema is below. Binary mutation columns are 1 = mutated, 0 = wild-type.
Quantile-derived `label_*` columns are binarised at the within-cohort 75th percentile (Q75)
unless noted; `NaN` = label unavailable for that patient.

| Column | Type | Description |
|--------|------|-------------|
| `patient_id` | str | 12-char TCGA barcode |
| `cohort` | str | TCGA study code (ACC, UVM, MESO, CHOL, THYM, KICH, DLBC) |
| `mut_<GENE>` | int | Somatic mutation status for the cohort's driver genes (1 mutated / 0 wild-type). Examples: `mut_TP53`, `mut_GTF2I`, `mut_GNAQ`, `mut_GNA11`, `mut_BAP1`, `mut_EIF1AX`, `mut_SF3B1`, `mut_CTNNB1`, `mut_MEN1`, `mut_NF2`, `mut_IDH1`, `mut_MYD88`, `mut_CD79B`, `mut_HRAS` |
| `msi_mantis` | float | MSI score (MANTIS) |
| `msi_sensor` | float | MSI score (MSIsensor) |
| `mutation_count` | float | Non-synonymous mutation count |
| `tmb` | float | Tumour mutational burden (mut/Mb) |
| `aneuploidy` | float | Aneuploidy score |
| `fga` | float | Fraction of genome altered |
| `histological_subtype` | str | Cohort-specific histological subtype text |
| `stage`, `grade`, `sex`, `age` | mixed | Clinical covariates (descriptive; not used as predictors) |
| `immune_subtype` | str | Thorsson pan-cancer immune subtype (C1–C6) |
| `label_tmb_high` | float | 1 if TMB ≥ within-cohort Q75 |
| `label_fga_high` | float | 1 if FGA ≥ within-cohort Q75 |
| `label_aneuploidy_high` | float | 1 if aneuploidy ≥ within-cohort Q75 |
| `label_msi_h` | float | MSI-high (DLBC: MANTIS ≥ 0.4; KICH: MSIsensor ≥ 3.5) |
| `label_chr3_loss` | float | UVM only — chromosome-3 loss (monosomy 3) |
| `label_histology_3class` | int | UVM only — 3-class histology label |
| `label_histology_epi` | float | MESO only — epithelioid (1) vs non-epithelioid (0) |
| `label_cdkn2a_homdel` | float | MESO only — CDKN2A homozygous deletion (GISTIC) |
| `label_hans_subtype` | str | DLBC only — Hans IHC class (GCB / non-GCB) |
| `label_hans_nongcb` | float | DLBC only — non-GCB (1) vs GCB (0) |

`missingness_report.csv` / `.parquet`: per-cohort, per-label count of available vs missing values.

## `data/splits/<cohort>_splits.csv`

| Column | Type | Description |
|--------|------|-------------|
| `patient_id` | str | 12-char TCGA barcode |
| `fold` | int | Cross-validation fold assignment (0–4) |

`split_metadata.json`: seed (42), `n_folds` (5), generation timestamp, per-cohort
stratification target, patient counts, fold sizes, and a SHA-256 prefix of each fold file
for integrity verification.

## `results/oof_predictions.csv`

Per-patient out-of-fold predicted probabilities from the linear probe — the values that
underpin the pooled AUROC/AUPRC, bootstrap CIs, and calibration analyses. Regenerated by
`src/export_oof_predictions.py` and verified to reproduce the published per-cell AUROCs.

| Column | Type | Description |
|--------|------|-------------|
| `cohort` | str | TCGA study code |
| `biomarker` | str | Biomarker name (e.g. `GTF2I`, `MSI-H`) |
| `tier` | str | `primary` or `exploratory` |
| `patient_id` | str | 12-char TCGA barcode |
| `y_true` | int | Ground-truth label (1 positive / 0 negative) |
| `oof_proba` | float | Out-of-fold predicted probability of the positive class |
| `fold` | int | Test fold the prediction came from (0–4) |

## `results/abmil_attention.json`

Top-attention patches from the gated-attention ABMIL model for the five FDR-significant
cells (`DLBC/MSI-H`, `THYM/GTF2I`, `THYM/TMB-high`, `UVM/EIF1AX`, `UVM/Chr3 loss`). Keyed by
`"<cohort>/<biomarker>"`; each cell has a `patients` list. Per patient:

| Field | Description |
|-------|-------------|
| `patient_id` | 12-char TCGA barcode |
| `true` | Ground-truth label |
| `pred` | ABMIL out-of-fold predicted probability |
| `correct` | 1 if the thresholded prediction is correct |
| `top_attn.coords` | Whole-slide (x, y) coordinates of the top-20 attended patches |
| `top_attn.weights` | Attention weight for each of those patches |
| `top_attn.n_patches` | Total number of patches in the slide |

## `results/*.json` (analysis outputs)

`phase3_baseline.json` (baseline gate), `phase4_benchmark.json` (per-cell metrics + CIs),
`phase4_multiclass.json` (multiclass positive controls), `calibration.json` (ECE/Platt/
isotonic), `permutation_tests.json` (permutation p + BH-FDR), `sensitivity_analyses.json`,
`s2_meanvar.json`, `site_analyses.json`, `failure_modes.json`, `phase5_transfer.json`
(TP53 transfer), `tmb_transfer.json` (TMB-high transfer), `phase5b_gigapath.json`
(UNI2-h vs Prov-GigaPath vs ensemble), `abmil_results.json` (ABMIL vs probe). Each maps
to the table/figure indicated in the README *Outputs* section.
